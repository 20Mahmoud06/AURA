# ✨ Overview

AURA is a modern Android media player built with Flutter, designed to provide a fast, immersive, and elegant entertainment experience.

Whether you're listening to music, watching videos, organizing playlists, or browsing local media libraries, AURA combines powerful playback capabilities with a cinematic user interface inspired by modern glassmorphism and premium media applications.

Built with clean architecture principles, modern Flutter technologies, and a strong focus on performance and user experience.

# 📦 Installation

Clone the repository:

```bash
git clone https://github.com/20Mahmoud06/AURA.git
```

Navigate to the project directory:

```bash
cd AURA
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
